Service Operations

TopDownSimpleService

Contains 4 service operations, importMarketData, downloadFile, convertMarketData and showMarketData.

  • importMarketData - Gathers all information about a specified security given a security code (sec), start date, end date and a data source to retrieve from. Our data source defaults to http://real-chart.finance.yahoo.com/table.csv. Saves the file to a retrievable location.
  • downloadFile - creates a downloadable link to the generated CSV file created from importMarketData, takes in an event set ID, which is the name of the generated file.
  • convertMarketData - takes an existing security file with a specific eventsetid and generated by importMarketData and converts all monetary values within the file into the chosen the currency.
  • showMarketData - takes 2 existing security files and use a chart javascript to show the results.

Error codes

TopDownSimpleService

importMarketData

  • CODE: ERR_NULL_VALUE
  • MSG: One of the input is null
  • CODE: ERR_SEC
  • MSG: SEC code should be 3 to 6 characters long
  • CODE: ERR_DATE
  • MSG: Date format should be DD-MM-YYYY
  • CODE: ERR_RANGE
  • MSG: Range format should be d/w/m/v
  • CODE: ERR_DATE
  • MSG: Start dat should before end date

downloadFile

  • CODE: ERR_EVENTSET_ID
  • MSG: EventSetId not exist

convertMarketData

  • CODE: ERR_EVENTSET_ID
  • MSG: EventSetId not exist
  • CODE: ERR_CURRENCY
  • MSG: Target currency is not available

showMarketData

  • CODE: ERR_EVENTSET_ID
  • MSG: EventSetId not exist
  • CODE: ERR_NOT_COMPARABLE
  • MSG: The two files should have same currency and monthly range
